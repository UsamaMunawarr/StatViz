🏠

Home Dashboard

Welcome to StatViz. This Dashboard serves as your central hub to navigate through the entire data analysis platform effortlessly, without writing a single line of code.

📂

1. Data Import & Edit

Upload your datasets with ease via CSV or Excel mapping. You also have the powerful ability to edit your raw data directly within the platform before analysis.

🔀

2. Data Preparation

Comprehensive data wrangling: Apply data type conversions, encode variables, standardize/scale data, delete or rename columns, and securely save your prepped dataset back to CSV or Excel.

🔍

3. Filtering & EDA

Filter records dynamically and perform Exploratory Data Analysis. Handle missing data through visualizations and tackle outliers with advanced Winsorization.

📊

4. Statistics & Plotting

Compute deep descriptive statistics out of the box. Generate interactive, fully responsive visual charts to uncover hidden data patterns in seconds.

🧠

5. Hypothesis Testing

Run rigorous statistical tests seamlessly. Test for Normality, and execute Parametric (ANOVA, T-Tests) or Non-Parametric tests instantly.

📐

6. Multivariate Analysis

Advanced Dimensionality Reduction. Digest complex variable relationships utilizing Principal Component Analysis (PCA) and Exploratory Factor Analysis (EFA).

🤖

7. Statistical & ML Modeling

  • Advanced Algorithms: SVR/SVC, Random Forest, XGBoost, KNN, and Artificial Neural Networks (ANN) with hyperparameter tuning.
  • SPSS-Style Engine: Formatted output tables for Linear & Logistic Regression (Unstandardized B, Std. Error, Beta, p-values).
  • Variable Selection: Automated Forward, Backward, and Stepwise data selection methods.
  • Feature Importance: Deep visual analysis using both Impurity-based & Permutation techniques.
  • Export & Reports: One-click download of a compiled Model Summary PDF Report!
📈

8. Time Series

Forecast and predict underlying trends dynamically with custom auto-ARIMA modeling applied directly to your date-indexed variables.

🖨️

9. Generate PDF Reports

Automate your documentation. Compile all insights and model analysis down to a single cohesive, fully downloadable PDF Statistical Report.
